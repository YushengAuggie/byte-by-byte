### Issue
The quiz question asks "After clicking +1 twice, what does 'Renders' show?" with answer **C) 3**.

This is correct in production mode. However, **React 18 Strict Mode** (enabled by default in `create-react-app` and Next.js development builds) double-invokes render functions intentionally — so the actual count in a typical dev environment would be **5** (initial render runs twice = 2, each click runs twice = 2+2, but only one increment per render call... actually 5: 2 initial + 2 click1 + ... let me be precise).

In React 18 Strict Mode dev:
- Initial mount: renders twice → renderCount.current = 2
- Click 1: renders twice → renderCount.current = 4
- Click 2: renders twice → renderCount.current = 6

So the displayed value would be 6, not 3, in a standard dev setup.
