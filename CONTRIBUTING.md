# Contributing to byte-by-byte

Thanks for your interest! Here's how you can help.

## Ways to Contribute

### 📝 Content
- **Add problems** — extend `content/neetcode-150.json` or other topic databases
- **Improve existing content** — fix errors, improve explanations in `samples/` or `archive/`
- **Translate** — add more languages beyond Chinese/English
- **Add "further reading" links** — curate the best external resources for each topic

### 🐛 Bug Reports
- Found a wrong answer in an algorithm? Open an issue with the expected vs actual output
- QA report flagged something? Help verify and fix it

### 💡 Feature Ideas
- New sections (database, DevOps, etc.)
- Alternative delivery methods (Discord bot, email digest, RSS)
- Spaced repetition / Anki export
- GitHub Actions for non-OpenClaw users

## Content Quality Bar

If you're writing content, follow these guidelines:

- **Bilingual** — Chinese first, English second
- **Teach, don't lecture** — use real-world analogies
- **Code must be correct** — trace through your examples mentally
- **Diagrams help** — ASCII art for system design, visual traces for algorithms
- **Be interesting** — would YOU read this with your morning coffee?

## How to Submit

1. Fork the repo
2. Create a branch (`git checkout -b improve-day5-algo`)
3. Make your changes
4. Test if applicable (`bash scripts/generate.sh`)
5. Submit a Pull Request

## Code of Conduct

Be kind. Be helpful. We're all here to learn.
